## 1. 프로젝트 개요

### 1.1 분석 주제

**"넷플릭스 흥행작 '케이팝 데몬 헌터스(케데헌)'의 온라인 관심도 상승이 국립중앙박물관 관람객 수요에 미친 영향 분석"**

→ 이를 기반으로 **박물관 운영 정책(예약제 유효화 등) 의사결정 근거** 도출

---

### 1.2 비즈니스 맥락

#### **국립중앙박물관 수익 구조:**

- 입장료: 무료
- 주요 수익원: 문화상품(뮷즈) 판매, 정부 예산
- 가치 창출: 문화 향유 확대 → 정부 예산 정당성, 브랜드 가치

#### **케데헌 효과 분석의 의의:**

1. K-콘텐츠가 전통문화 관심으로 전환되는지 검증
2. 향후 넷플릭스 등 콘텐츠 기업과 협업 전략 수립
3. 급증한 관람객 대응 방안 (예약제 등) 마련

## 3. 핵심 가설 3개

### H1. 케데헌 개봉 이후 관람객 수가 유의미하게 증가했다 (인과관계)

**검증 방법:**

- 전년 동월 대비(YoY) 증감률 계산
- 케데헌 개봉 전후(2025.1-6월 vs 7-12월) 평균 비교
- 통계적 유의성 검증 (T-test, p < 0.05)

**예상 결과:**

- 2025년 7월: 747,679명 (전년 대비 **+106.8%**)
- 2025년 8월: 864,977명 (전년 대비 **+102.0%**)

---

### H2. 관람객 증가는 일시적 이벤트가 아니라 구조적 상승이다 (지속성)

**검증 방법:**

- 혼잡 지수 = (해당 월 관람객) / (2025년 월평균)
- 개봉 후 3개월(7~9월) 평균 vs 개봉 전 평균 비교
- 일시 이벤트: 7월만 급증 → 8~9월 원상복귀
- 구조 상승: 7~9월 모두 높은 수준 유지

**예상 결과:**

- 개봉 후 3개월 평균 혼잡 지수 **1.5 이상** 유지

---

### H3. 관람객 증가는 뮷즈 매출 증가로 이어졌다 (경제적 임팩트)

**검증 방법:**

- ARPU = 월별 뮷즈 매출(백만원) / 월별 관람객 수(명) × 1,000,000
- 상관분석: 관람객 수 vs 뮷즈 매출 (Pearson r)
- 케데헌 개봉 전후 ARPU 비교

**예상 결과:**

- 2025년 7월 뮷즈 매출: 4,957백만원 (전년 대비 **+180.7%**)
- 상관계수 r > 0.7 (강한 양의 상관)

### 5.3 보조 지표 (4개)

#### **1. 전년 동월 대비 관람객 증감률 (YoY, Year-over-Year)**

**산출식:**

```
YoY(%) = (2025년 X월 관람객 수 - 2024년 X월 관람객 수) / 2024년 X월 관람객 수 × 100
```

**활용:**

- 케데헌 개봉 전후 증가율 비교
- 계절성 효과 제거 (전년 동월 대비)

**예시:**

```
2025년 7월: 747,679명
2024년 7월: 361,493명
YoY = (747,679 - 361,493) / 361,493 × 100 = 106.8%
```

---

#### **2. 내·외국인 관람객 수 및 비중**

**산출식:**

```
외국인 비율(%) = 외국인 관람객 수 / 총 관람객 수 × 100
내국인 비율(%) = 내국인 관람객 수 / 총 관람객 수 × 100
```

**활용:**

- 케데헌이 글로벌 콘텐츠이므로 외국인 증가 예상
- 내국인 = 네이버 검색량, 외국인 = Google 검색량 상관관계

**예시:**

```
2025년 7월:
- 총 관람객: 747,679명
- 외국인: 26,730명 (3.57%)
- 내국인: 720,949명 (96.43%)
```

---

#### **3. 혼잡 지수 (월별 관람객 ÷ 기준 평균)**

**산출식:**

```
혼잡 지수 = 해당 월 관람객 수 / 2025년 월평균 관람객 수
```

**해석:**

- 1.0: 평균 수준
- 1.5 이상: 평균 대비 50% 이상 혼잡
- 2.0 이상: 매우 혼잡 (예약제 필요성)

**활용:**

- 구조적 상승 vs 일시적 이벤트 구분
- 예약제 도입 타당성 근거

**예시:**

```
2025년 월평균: 542,290명
2025년 7월: 747,679명
혼잡 지수 = 747,679 / 542,290 = 1.38 (평균 대비 38% 혼잡)
```

---

#### **4. 방문객 1인당 뮷즈 구매액 (ARPU, Average Revenue Per User)**

**산출식:**

```
ARPU(원) = 월별 뮷즈 매출액(백만원) / 월별 관람객 수(명) × 1,000,000
```

**활용:**

- 관람객 증가 시 실제 소비도 증가했는지 검증
- 단순 방문 vs 적극적 소비 구분
- 뮷즈 매출 예측 모델

**예시:**

```
2025년 7월:
- 뮷즈 매출: 4,957백만원
- 관람객 수: 747,679명
ARPU = 4,957,000,000 / 747,679 = 6,630원
```

## 6. 분석 방법론

### 6.1 2단계 분석 구조

#### **분석 1층: 거시적 맥락 (6개년 연간 데이터)**

**목적:** "2025년 급증이 얼마나 이례적인가?" 증명

**분석 방법:**

1. **CAGR (연평균 성장률) 계산**

   ```
   CAGR = (마지막값 / 첫값) ^ (1/기간) - 1

   2019~2024년 CAGR 계산
   예상치: 2025년 = 2024년 × (1 + CAGR)
   실제치: 2025년 = 650만 명

   → 예상치 vs 실제치 차이 = 케데헌 효과
   ```
2. **추세선 분석**

   - 2019~2024년 선형 회귀
   - 2025년 값이 추세선을 얼마나 벗어나는지 측정

**Tableau 시각화:**

- 라인 차트: 2019~2025 연간 관람객 추이
- 추세선(Trend Line) 표시
- 2025년 값 강조 (색상, 크기)

---

#### **분석 2층: 케데헌 효과 (2개년 월별 데이터)**

**목적:** "케데헌 개봉 전후 변화" 측정

**분석 방법:**

1. **전년 동월 대비 (YoY) 비교**

   ```
   2024년 1~12월 vs 2025년 1~12월
   → 7~8월 급증 확인
   ```
2. **개봉 전후 평균 비교**

   ```
   개봉 전 (2025.1~6월) 평균: A명
   개봉 후 (2025.7~12월) 평균: B명
   증가율 = (B - A) / A × 100
   ```
3. **통계적 유의성 검증 (T-test)**

   ```
   H0: 개봉 전후 관람객 평균 차이 없음
   H1: 개봉 후 관람객 평균이 유의미하게 높음
   유의수준: p < 0.05
   ```

**Tableau 시각화:**

- 라인 차트: 2024 vs 2025 월별 추이
- 케데헌 개봉일 (6/20) 수직선 마커
- YoY 증감률 막대 차트

### 6.2 상관분석

#### **상관분석 1: 검색량 vs 관람객**

**분석:**

```python
# Pearson 상관계수
correlation_google = corr(Google검색량, 외국인관람객수)
correlation_naver = corr(네이버검색량, 내국인관람객수)
```

**예상 결과:**

- r > 0.7: 강한 양의 상관
- r = 0.5~0.7: 중간 정도 상관
- r < 0.5: 약한 상관

**시차 분석 (Lag Analysis):**

```python
# 검색 후 N주 뒤 관람 가설
for lag in [1, 2, 3, 4]:  # 1~4주 시차
    corr = correlation(검색량, 관람객.shift(lag))
    print(f"{lag}주 시차 상관계수: {corr}")
```

---

#### **상관분석 2: 뮷즈 매출 vs 관람객**

**분석:**

```python
correlation_arpu = corr(관람객수, ARPU)
correlation_sales = corr(관람객수, 뮷즈매출)
```

**해석:**

- 상관계수 높음 → 관람객 증가가 매출로 직결
- 상관계수 낮음 → 관람객은 늘었지만 소비는 안 함

### 6.3 데이터 전처리 (Python)

```python
import pandas as pd
import numpy as np

# 1. 데이터 로드
visitors = pd.read_csv('monthly_visitors.csv')
google_trends = pd.read_csv('google_trends.csv')
naver = pd.read_excel('naver_datalab.xlsx')
netflix = pd.read_csv('netflix_top10.csv')
youtube = pd.read_excel('youtube_manual.xlsx')

# 2. 날짜 형식 통일
visitors['연월'] = pd.to_datetime(visitors['연도'].astype(str) + '-' + visitors['월'].astype(str))
google_trends['연월'] = pd.to_datetime(google_trends['Week']).dt.to_period('M')

# 3. 주간 → 월간 집계
google_monthly = google_trends.groupby('연월').mean()
naver_monthly = naver.groupby('연월').mean()

# 4. 데이터 조인
df = pd.merge(visitors, google_monthly, on='연월', how='left')
df = pd.merge(df, naver_monthly, on='연월', how='left')

# 5. 계산된 필드 생성
df['YoY'] = df.groupby('월')['총관람객'].pct_change(12) * 100
df['외국인비율'] = df['외국인'] / df['총관람객'] * 100
df['혼잡지수'] = df['총관람객'] / df['총관람객'].mean()

# 6. ARPU 계산
sales = pd.read_csv('goods_monthly_sales.csv')
df = pd.merge(df, sales, on='연월')
df['ARPU'] = df['매출액_백만원'] * 1000000 / df['총관람객']

# 7. 케데헌 개봉 전후 더미 변수
df['케데헌_개봉후'] = (df['연월'] >= '2025-06-20').astype(int)

# 8. CSV 저장
df.to_csv('final_dataset.csv', index=False, encoding='utf-8-sig')
```

### 10.2 통계 분석 참고

**T-test (독립표본 t-검정):**

```python
from scipy import stats

group1 = df[df['케데헌_개봉후'] == 0]['총관람객']  # 개봉 전
group2 = df[df['케데헌_개봉후'] == 1]['총관람객']  # 개봉 후

t_stat, p_value = stats.ttest_ind(group1, group2)
print(f"t-statistic: {t_stat}")
print(f"p-value: {p_value}")

if p_value < 0.05:
    print("귀무가설 기각: 개봉 후 관람객이 유의미하게 증가")
else:
    print("귀무가설 채택: 유의미한 차이 없음")
```

**상관분석 (Pearson Correlation):**

```python
correlation = df[['Google검색량', '외국인관람객']].corr()
print(correlation)

# 해석
# r > 0.7: 강한 양의 상관
# r = 0.5~0.7: 중간 정도 상관
# r < 0.5: 약한 상관
```
